Python3_random模块的方法详解

2019年3月5日 发布在 Python学习笔记

描述

random() 方法返回随机生成的一个实数，它在(0,1)范围内。

语法

以下是 random() 方法的语法:

import random

random.random()

注意：random()是不能直接访问的，需要导入 random 模块，然后通过 random 静态对象调用该方法。
